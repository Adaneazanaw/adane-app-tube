# Adane App Tube

A premium full-stack tech news platform.

## Key points

- Full-stack app with a Node.js backend and a browser frontend
- Backend entry point: `Backend/server.js`
- Frontend entry point: `Frontend/app.js`
- Uses `nodemon` for development hot-reloading

## Install

- Run `npm install` in the project root
- Then run `npm install` again inside `Backend`

## Run

- Start the app with `npm start`
- For development use `npm run dev`

## Project structure

- `Backend/` - server code and backend logic
- `Frontend/` - browser UI code
- `styles/` - CSS files for layout and design

## Notes

- `npm install-all` installs dependencies in both root and `Backend`
- `main` points to `Backend/server.js`
